# Automatischer Datei-Sortierer (Python)

Ein Automatisierungsskript, das unaufgeräumte Ordner analysiert und Dateien basierend auf ihrer Dateiendung automatisch in entsprechende Unterordner sortiert. 

### Funktionen
- Automatische Erstellung von Zielordnern (Bilder, Dokumente, Videos, Sonstiges)
- Erkennung von Dateitypen anhand der Dateiendung (.pdf, .jpg, .mp4, etc.)
- Verschieben der Dateien in Echtzeit mit detaillierter Konsolen-Ausgabe

### Technische Highlights
- Nutzung der integrierten Python-Bibliotheken `os` und `shutil`
- Manipulation und Validierung von Dateipfaden (Path-Handling)
- Sichere Systemoperationen zur Vermeidung von Datenverlust

### Verwendete Technologien
- Python 3.x
- System-Automatisierung
